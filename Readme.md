# Loulib

---
## Desc
Common Helper For Android.

## How to use?
add below code to `app/build.gradle`
```
dependencies {
    compile fileTree(include: ['*.jar'], dir: 'libs')
    compile 'com.lyloou:lou:1.0.0'
}
```
